This corrupted the shared dev DB's `migrations` tracking table and the `videos` FK constraint twice during verification of this SI; both times repaired manually via SQL (re-inserting migration tracking rows, re-adding the FK constraint, clearing stray rows). Flagged as a separate follow-up task (`task_c6538a77`) rather than fixed inline — out of scope for Video entity work, and touches Phase 2 test infrastructure.
